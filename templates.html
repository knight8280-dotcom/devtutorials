Threat & Aggro System

Tank mechanics that keep enemies focused on defenders.

Code: SYS-TA-002

// Increase threat on an enemy
void addThreat(Enemy enemy, float amount) {
    enemy.threat += amount;
    sortThreatList();
}

PvP Toggle & Ranking

Allow players to toggle PvP mode and track their rank.

Code: SYS-PVP-003

bool pvpEnabled = false;
void togglePvP() {
    pvpEnabled = !pvpEnabled;
    updateRankings();
}

Raid Encounter System

Create multi‑phase boss fights with scripted phases.

Code: SYS-RA-004

void startPhase(int phase) {
    switch (phase) {
        case 1: spawnAdds(); break;
        case 2: unleashBossAbility(); break;
        case 3: triggerEnrage(); break;
    }
}

Inventory & Loot System

Manage inventories and random loot drops from enemies.

Code: SYS-IN-005

Item dropRandomLoot() {

Crafting & Farming

Create farming plots and recipes for crafting items.

Code: SYS-CR-006

bool craft(Item[] ingredients, Recipe recipe) {
    return recipe.matches(ingredients);
}
