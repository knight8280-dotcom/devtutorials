Game UI/UX Design

Code: TUT-UXD-001

$39.99

Course Overview

Master UI/UX design for games using UMG in Unreal Engine 5. Learn to create menus, HUDs, and interactive interfaces that enhance player experience.

What You'll Learn

  • UMG Fundamentals: Widgets, canvas panels, and layout
  • Design Principles: Visual hierarchy, readability, and usability
  • Menu Systems: Main menus, pause menus, and settings
  • HUD Design: Health bars, minimaps, and status displays
  • Interactive Elements: Buttons, sliders, and input fields
  • Animation: Widget animations and transitions
  • Responsive Design: Scaling for different resolutions
  • Accessibility: Colorblind modes and font sizing

What's Included

Includes: 15 hours video, UI templates, widget blueprints, project files, PDF guide (125 pages), community access

Prerequisites

  • Basic Unreal Engine knowledge recommended
  • Understanding of game development fundamentals
