Overview

A comprehensive combat system built on Unreal Engine 5's Gameplay Ability System (GAS). This production-ready template includes everything from basic attacks to complex ability combos, damage calculation, status effects, and combat stats management.

Problems This Solves

  • Ability System: Implement special attacks, spells, and abilities with cooldowns
  • Damage Calculation: Handle physical, magical, and elemental damage with armor/resistance
  • Status Effects: Apply buffs, debuffs, DoT, and crowd control effects
  • Combat Stats: Manage health, mana, stamina, and derived stats (crit, dodge, etc.)
  • Targeting System: Implement lock-on, area targeting, and smart target selection

Technical Specifications

  • Engine: Unreal Engine 5.0+ utilizing Gameplay Ability System (GAS)
  • Architecture: Component-based with modular ability system
  • Combat Model: Server-authoritative with client prediction for responsiveness
  • Networking: Fully replicated for multiplayer with lag compensation
  • Animation: Integrated with animation montages and root motion
  • Performance: Optimized ability execution with object pooling

Key Features

  • Gameplay Ability System (GAS) integration for abilities and effects
  • Attribute system: Health, Mana, Stamina, Attack Power, Defense, Critical Chance
  • Damage types: Physical, Magical, Fire, Ice, Lightning, Poison
  • Status effects: Stun, Root, Slow, Silence, Bleed, Burn, Freeze
  • Combo system with input buffering and canceling
  • Ability cooldown and resource cost management
  • Hit detection with hitbox visualization tools
  • Lock-on targeting with soft and hard lock modes
  • Damage numbers and combat feedback system
  • Threat/aggro system for AI enemy targeting
