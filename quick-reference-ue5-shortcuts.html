UE5.6 Keyboard Shortcuts

Complete reference guide - Speed up your workflow 10x!

Download PDF Version - Print this and keep it next to your desk!

Viewport Navigation

Action Shortcut Description
Rotate Camera Right Mouse + Drag Look around (hold and move mouse)
Pan Camera Middle Mouse + Drag Move camera up/down/left/right
Zoom Camera Right Mouse + Mouse Wheel Move camera forward/backward
Fly Camera (WASD) Right Mouse + W/A/S/D Fly around like FPS game
Fly Up/Down Right Mouse + E/Q Ascend (E) or Descend (Q)
Speed Up Movement Right Mouse + Mouse Wheel Up Fly faster (scroll up repeatedly)
Slow Down Movement Right Mouse + Mouse Wheel Down Fly slower (scroll down repeatedly)
Focus on Object F Center camera on selected object
Orbit Around Object Alt + Left Mouse Rotate camera around selected object
Zoom to Object Alt + Right Mouse Zoom in/out on selected object

Object Manipulation

Action Shortcut Description
Move Tool W Switch to translate (move) gizmo
Rotate Tool E Switch to rotate gizmo
Scale Tool R Switch to scale gizmo
Duplicate Object Alt + Drag Hold Alt while dragging to duplicate
Snap to Grid (Move) Ctrl + Drag Hold Ctrl while moving to snap
Snap to Rotation Ctrl + E Snap rotation to 10° increments
Drop to Floor End Snap object to ground
Move to Origin Home Reset object to 0,0,0
Toggle World/Local Space Ctrl + ` Switch between world and local coordinates

Edit Operations

Action Shortcut Description
Copy Ctrl + C Copy selected objects
Paste Ctrl + V Paste copied objects
Cut Ctrl + X Cut selected objects
Duplicate Ctrl + D Duplicate selected objects
Delete Delete Delete selected objects
Undo Ctrl + Z Undo last action
Redo Ctrl + Y Redo undone action
Select All Ctrl + A Select all objects in level
Deselect All Escape Clear selection

Play & Simulation

Action Shortcut Description
Play (PIE) Alt + P Play in Editor
Stop Play Escape Stop playing
Pause Pause Pause game
Eject from Player F8 Free camera while playing
Possess Player F8 Return to player control
Simulate Physics Ctrl + P Simulate without possessing player

Project & File

Action Shortcut Description
Save Current Level Ctrl + S Save current level/map
Save All Ctrl + Shift + S Save all modified assets
Open Level Ctrl + O Open existing level
New Level Ctrl + N Create new level
Content Browser Ctrl + Space Open/focus Content Browser
Quick Search Ctrl + P Quick asset search

Viewport Display

Action Shortcut Description
Wireframe Alt + 2 Show wireframe view
Unlit Alt + 3 Show without lighting
Lit Alt + 4 Show with lighting (default)
Detail Lighting Alt + 5 Show lighting complexity
Lighting Only Alt + 6 Show only lighting (no textures)
Show/Hide Grid G Toggle floor grid
Show/Hide Collision Alt + C Show collision meshes
Show FPS Ctrl + Shift + H Show framerate stats
Shader Complexity Alt + 8 Show performance heatmap

Blueprint Editor

Action Shortcut Description
Compile Blueprint F7 Compile current Blueprint
Comment Box C Create comment around selection
Find in Blueprint Ctrl + F Search nodes/variables
Find References Shift + F Find all references to selected
Toggle Breakpoint F9 Add/remove debug breakpoint
Delete Node Delete Delete selected nodes
Cut Wires Alt + Click Click wire to disconnect
Straighten Connection Q Straighten wire between nodes
Align Nodes Shift + W/E Align selected nodes

Camera Bookmarks

Action Shortcut Description
Set Bookmark 0-9 Ctrl + 0-9 Save camera position
Jump to Bookmark 0-9 0-9 Jump to saved position

Advanced Tips

Speed Boost Combo

Hold Right Mouse + W + scroll Mouse Wheel Up repeatedly = Fly at super speed through huge levels!

Quick Duplicate Line

Select object → Alt + Drag → Shift (hold both) = Duplicate in a perfect line

Precision Movement

While moving object, tap Ctrl repeatedly to nudge in small grid increments

Pro Tip: Customize Your Shortcuts!

Go to Edit → Editor Preferences → Keyboard Shortcuts to customize any shortcut to your liking!
