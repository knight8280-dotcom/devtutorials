Course Overview

Learn to create polished 2D games using Unreal Engine 5's Paper2D framework. This course covers sprite creation, animation, tilemaps, 2D physics, and gameplay mechanics specific to 2D game development.

What You'll Learn

  • Paper2D System: Complete mastery of UE5's 2D framework
  • Sprite Management: Import, flipbook animation, and sprite sheets
  • Tilemap Creation: Build levels with tile sets and auto-tiling
  • 2D Physics: Collision, gravity, and platformer mechanics
  • Character Animation: State machines for 2D character animation
  • Camera System: Side-scrolling and fixed camera setups
  • Parallax Scrolling: Multi-layer backgrounds for depth
  • 2D Lighting: Dynamic lighting and shadow effects

Course Structure

  • Module 1 - Paper2D Fundamentals (2 hours): Sprites, flipbooks, and basic setup
  • Module 2 - Tilemap System (2.5 hours): Creating levels with tiles
  • Module 3 - 2D Character Controller (3 hours): Movement and animations
  • Module 4 - 2D Physics & Collision (2 hours): Platformer mechanics
  • Module 5 - Game Mechanics (3 hours): Pickups, enemies, and interactions
  • Module 6 - Polish & Effects (2 hours): Particles, lighting, and juice
  • Bonus Project (3 hours): Complete 2D platformer game

What's Included

  • Video Content: 17.5 hours of tutorial videos
  • 2D Asset Pack: Sprites, tilesets, and animations
  • Project Files: Complete source files for all modules
  • Sprite Templates: Reusable character and object templates
  • PDF Guide: 120-page written companion
  • Community Access: Student Discord channel
  • Lifetime Updates: Free future content additions

Prerequisites

  • Basic Unreal Engine knowledge (Fundamentals course recommended)
  • Understanding of Blueprint basics
  • Graphics software experience helpful but not required

Who This Course Is For

  • Indie developers creating 2D games
  • Artists transitioning to game development
  • Developers familiar with other 2D engines (Unity, GameMaker)
  • Anyone passionate about 2D gaming experiences

Includes: 17.5 hours video, 2D asset pack, project files, sprite templates, PDF guide, community access, lifetime updates
